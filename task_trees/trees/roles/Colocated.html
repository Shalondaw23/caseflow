    Colocated Description #

    The Colocated user role is any user of the Caseflow application performing the duties of a VSJ support staff in the Veteran appeals process (see Colocated). Users with this role can complete various tasks in the appeals process to support the attorneys and judges with moving the appeal through its different phases. Users with this role can also have the Hearings Users or Intake Users roles.

    Tasks #

    There are many different types of tasks that can be completed by Colocated users depending on where the case is in the appeals process. For more information on how tasks are created through the different phases, see Tasks Overview.
